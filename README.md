# Path to Peace

**Path to Peace** is a digital sanctuary designed as a "Meditative Editorial." It offers a serene, high-end experience for spiritual seekers, blending modern design aesthetics with classical wisdom.

## 🌿 The Design Philosophy: "The Meditative Editorial"

The project follows a strict design system called "The Meditative Editorial," characterized by:
- **Intentional Asymmetry**: Layouts that breathe through expansive negative space.
- **Tonal Layering**: Depth achieved through subtle color shifts rather than heavy shadows.
- **The Emerald & Cream Palette**: A sophisticated interplay of deep greens (#003527) and luminous creams (#fbf9f5).
- **Elegant Typography**: A contrast between the soul-stirring *Noto Serif* and the architectural *Plus Jakarta Sans*.

## 🛠️ Technology Stack

- **Core**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui (Radix UI)
- **Icons**: Lucide React
- **Aesthetics**: Glassmorphism, Custom SVG Star Patterns, and Tonal Elevation.

## 📂 Project Structure

- `app/`: Next.js App Router routes (Home, About, Contact).
- `components/layout/`: Shared layout elements like `Navbar` and `Footer`.
- `components/sections/`: High-level landing page sections (Hero, Guidance, CTA).
- `components/ui/`: Atomic UI components, including custom ones like `AyahBlock`.
- `public/`: High-resolution imagery and assets.

## 🚀 Getting Started

First, install the dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to enter the sanctuary.

## ✨ Key Features

- **Dynamic Hero Section**: Featuring serene mosque architecture and a "Wisdom of the Day" card.
- **Guidance from the Source**: A grid of spiritual wisdom featuring custom "Ayah Blocks."
- **Cultivate Your Inner Garden**: Interactive call-to-action sections with glassmorphism effects.
- **Responsive & Meditative**: Optimized for all devices with a focus on visual tranquility.
